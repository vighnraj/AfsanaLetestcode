================================================================================
                           PRODUCT OVERVIEW
================================================================================

EduCRM React Admin Dashboard is a comprehensive, modern admin panel template
built for educational institutions, universities, and counseling agencies.

Features:
- Student Management & Profiles
- Lead & Inquiry Management
- Admission Tracking & Visa Processing
- Task & Calendar Management
- Communication & Chat System
- Reporting & Analytics Dashboard
- Role-Based Access Control
- Multi-Role Support (Admin, Counselor, Staff, Student, Processor)

================================================================================
                        QUICK INSTALLATION
================================================================================

1. Navigate to the source folder:
   cd source

2. Install dependencies:
   npm install

3. Configure environment:
   - Copy .env.example to .env
   - Update the API URL and other settings

4. Start development server:
   npm run dev

5. Build for production:
   npm run build

For detailed instructions, see: documentation/installation.md
