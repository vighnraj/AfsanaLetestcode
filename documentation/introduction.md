# EduCRM React Admin Dashboard - Introduction

## Overview

EduCRM React Admin Dashboard is a comprehensive, modern, and feature-rich **frontend-only** admin panel template built specifically for educational institutions, universities, and counseling agencies. This template provides a complete solution for managing students, leads, admissions, visa processing, and more.

## Important Notice

**This is a FRONTEND-ONLY template.** The backend/API is NOT included in this package. You are responsible for:
- Setting up your own backend server
- Creating API endpoints
- Database configuration
- Authentication system

## Key Features

### Student Management
- Student profiles and details
- Application tracking
- Document management
- Visa processing workflow

### Lead & Inquiry Management
- Lead capture and tracking
- Inquiry management
- Follow-up scheduling
- Status tracking

### Admission Tracking
- Application timeline
- University submissions
- Admission decisions
- Counselor applications

### Task & Calendar Management
- Task assignment
- Task reminders
- Calendar integration
- Staff management

### Communication
- Real-time chat
- Automated reminders
- Follow-up scheduling
- Support system

### Reporting & Analytics
- Dashboard analytics
- Payment reports
- Branch management
- Invoice generation

### User Management
- Role-based access control
- Permission management
- Multi-role support (Admin, Counselor, Staff, Student, Processor)

## Technology Stack

- **React 18** - Frontend library
- **Vite** - Build tool and dev server
- **React Router DOM 7** - Client-side routing
- **Bootstrap 5** - CSS framework
- **Material UI** - Component library
- **Firebase** - Optional authentication and notifications
- **Axios** - HTTP client
- **Chart.js** - Data visualization
- **FullCalendar** - Calendar component
